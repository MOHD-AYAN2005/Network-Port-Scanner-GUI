# Network Port Scanner GUI

## Overview
The Network Port Scanner GUI is a user-friendly application designed to scan and analyze open ports on a specified network. This tool is particularly useful for system administrators and security professionals to identify active services and potential vulnerabilities.

## Features
- **User-Friendly Interface**: Intuitive graphical interface for easy operation.
- **Port Scanning**: Ability to scan a range of ports to detect open services.
- **Service Identification**: Identifies services running on open ports.
- **Export Results**: Option to save the results in various formats.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/MOHD-AYAN2005/Network-Port-Scanner-GUI.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Network-Port-Scanner-GUI
   ```

3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Launch the application:
   ```bash
   python main.py
   ```

2. Enter the target IP address and the range of ports to scan.
3. Click on the ‘Scan’ button to begin.
4. Review the open ports and associated services displayed in the results section.
